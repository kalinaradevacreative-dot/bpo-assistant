# BPO Performance Assistant 🚀

A Python-based productivity tool designed for BPO (Business Process Outsourcing) environments.

## Features
- **Shift Progress Tracker:** Monitors how much of the 8-hour shift is remaining.
- **KPI Calculator:** Automatically calculates productivity scores based on resolved cases.
- **Real-time Feedback:** Notifies the user if they are meeting their hourly targets.

## How to Run
1. Make sure you have Python installed.
2. Run the script via Terminal:
   ```bash
   python3 bpo_tool.py
   ```
